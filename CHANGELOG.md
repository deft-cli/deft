# Changelog

All notable changes to this project.

## [0.2.1] - 2026-06-23

- Add CI workflow for testing across multiple OS environments.
- Add tests for `sync` and `update` subcommands.
- Add funding configuration.

## [0.2.0] - 2026-06-22

- Add full-featured CLI with `build`, `sync`, `update`, `doctor`, and `migrate` commands.
- Add core build engine with parallel compilation.
- Add dependency resolver and package index sync.
- Add manifest and lockfile data models.
- Add C/C++ build argument generation.
- Add centralized error handling.
- Add `migrate --from=cmake` command to import existing CMake projects.

## [0.1.0] - 2026-06-16

- Initial release with core functionality.
